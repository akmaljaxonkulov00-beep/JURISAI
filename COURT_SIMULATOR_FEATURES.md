## 🚀 API ENDPOINTS

### 1. POST /api/court-simulator
```typescript
// Start simulation
{
  action: 'start',
  caseDetails: string
}

// Submit argument
{
  action: 'submit_argument',
  simulationId: string,
  argument: string
}

// Get verdict
{
  action: 'get_verdict',
  simulationId: string
}
```

### 2. GET /api/court/cases
```typescript
// Get all cases
Response: {
  cases: Case[],
  total: number
}
```

### 3. GET /api/court/history
```typescript
// Get user history
Response: {
  history: SimulationHistory[],
  stats: UserStats
}
```
