│  🏛️ COURT SIMULATOR TEST                                     │
└──────────────────────────────────────────────────────────────┘

URL: http://localhost:3000/court-simulator

Workflow:
  1. "Holatlar" → Holat tanlash → "Simulyatsiyani boshlash"
  2. "Simulyatsiya" tabiga o'tadi
  3. "🎤 Gapiring" tugmasini bosing
  4. Browser ruxsat so'rasa → "Allow"
  5. Mikrofonga gapiring:
     "Mening mijozim aybsiz, dalillar etarli emas"
  6. Matn avtomatik yoziladi (real-time)
  7. "📤 Argumentni yuborish" bosing
  8. AI javobi ovozda eshitiladi ✅

Tugmalar:
  • 🎤 Gapiring - Mikrofon yoqish/o'chirish
  • 🔊 Eshitish ON - AI javoblarni eshitish
  • 🔇 Eshitish OFF - Faqat yozma
  • ⏹️ To'xtatish - Ovozni to'xtatish

┌──────────────────────────────────────────────────────────────┐
│  ⚠️ MUAMMOLAR VA YECHIMLAR                                   │
└──────────────────────────────────────────────────────────────┘

1. Mikrofon ishlamayotgan bo'lsa:
   • Chrome Settings → Privacy → Microphone → Allow
   • Windows Settings → Privacy → Microphone → ON
   • Mikrofon ulanganligini tekshiring
   • Sahifani yangilang (F5)

2. Ovoz chiqmayotgan bo'lsa:
   • "🔊 Eshitish ON" tugmasi yashil rangda bo'lsin
   • Kompyuter ovozi o'chirilganmi?
   • Browser tab muted emas?
   • Console da error bormi?

3. Matn tanilmayotgan bo'lsa:
   • Chrome yoki Edge ishlatayapsizmi?
   • Internetingiz bormi? (STT online)
   • Aniq va sekin gapiring
   • Rus tilida gapiring (yaxshiroq taniladi)

4. "Not Allowed" error:
   • Browser ruxsat berilmaganligini bildiradi
   • URL yonidagi 🔒 belgini bosing
   • Microphone → Allow
   • Sahifani yangilang
