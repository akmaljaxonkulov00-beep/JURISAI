┌──────────────────────────────────────────────────────────────┐
│  ⚠️ MUAMMOLAR VA YECHIMLAR                                   │
└──────────────────────────────────────────────────────────────┘

1. Ovoz chiqmayapti:
   DIAGNOSTIKA:
     Console: tts.getVoices().length
     Natija: 0 → Kutib turing (1-2 soniya)
     Natija: 12+ → Voices yuklangan ✅
   
   YECHIM:
     - Sahifani yangilang (Ctrl+R)
     - Ovoz balandligini tekshiring
     - Browser tab muted emasligini tekshiring

2. Mikrofon ishlamayapti:
   DIAGNOSTIKA:
     Console:
       navigator.mediaDevices.getUserMedia({audio: true})
         .then(() => console.log('Mic OK'))
         .catch(e => console.error(e))
   
   YECHIM:
     - URL → 🔒 → Microphone → Allow
     - Windows → Settings → Privacy → Microphone → ON
     - Mikrofon ulanganligini tekshiring

3. Matn tanilmayapti:
   DIAGNOSTIKA:
     Console da error bormi?
     "no-speech" → Aniqroq gapiring
     "not-allowed" → Ruxsat bering
   
   YECHIM:
     - Aniq va sekin gapiring
     - Rus tilida gapiring (yaxshiroq taniladi)
     - Shovqinsiz joyda test qiling

4. API 500 Error:
   DIAGNOSTIKA:
     Console → Network tab
     Check request payload
   
   YECHIM:
     - .env.local tekshiring:
       GROQ_API_KEY=gsk_ftEyNP...
     - Serverni qayta ishga tushiring

5. Demo data ko'rsatilmoqda:
   YECHIM:
     - Cache tozalang:
       rmdir /s /q .next
     - Serverni qayta ishga tushiring:
       npm run dev
